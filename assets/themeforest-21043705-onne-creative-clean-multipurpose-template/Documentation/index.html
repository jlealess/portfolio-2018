How to insert your images

1. Select the “bg” layer

02. Put your image above the “bg” layer

02. Select your image layer, right click and select “Create Clipping Mask”. Or you can use the hot keys Cmd / Ctrl + Shift + G

Change Opacity

Some images need to be darken or lighten to show the text clearly. Select “opacity” option and change its value to fit your image.
